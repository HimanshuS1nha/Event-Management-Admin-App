# Event Management Admin App

This is a part of the full stack event management app. Control everything from participants to events.

## Features

- Login/Logout
- Add, edit and delete participants, heads, scanners and events.

## TECH STACK USED

- React Native Expo
- Tailwind CSS
- Zustand
- Zod
